## Installation

To access the MongoDB Cheat Sheet, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/AliyanRizwanDev/MongoDB-EX.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd MongoDB-EX
   ```

3. **Open the PDF file:**
